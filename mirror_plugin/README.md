# mirror_plugin

Плагин отражения для системы `image_processor`. Выполняет горизонтальное и/или
вертикальное зеркальное отражение изображений в формате RGBA8.

## Алгоритм

Отражение реализовано **in-place** — без дополнительного буфера — попарным обменом
пикселей. Сначала применяется горизонтальное отражение (`flip_h`), затем
вертикальное (`flip_v`). Оба флага независимы и могут использоваться одновременно.

- **`flip_h`** — меняет местами левую и правую половины каждой строки (лево ↔ право).
- **`flip_v`** — меняет местами верхнюю и нижнюю половины изображения (верх ↔ низ).

## Параметры (`params.yaml`)

```yaml
flip_v: true    # вертикальное отражение (верх ↔ низ)
flip_h: true    # горизонтальное отражение (лево ↔ право)
```

Файл передаётся хосту через флаг `--params`. При некорректном YAML плагин
применяет `flip_v: false, flip_h: false` (изображение не изменяется).

## Использование

```bash
# Сборка плагина
cargo build -p mirror_plugin

# Применение через image_processor
cargo run -p image_processor -- \
  --input photo.png \
  --output mirrored.png \
  --plugin mirror_plugin \
  --params mirror_plugin/params.yaml
```

### Публичный C-ABI интерфейс

```rust

pub unsafe extern "C" fn process_image(
    width: u32,
    height: u32,
    rgba_data: *mut u8,
    params: *const c_char,
)
```
