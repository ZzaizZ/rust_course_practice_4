# blur_plugin

Плагин размытия для системы `image_processor`. Реализует взвешенное круговое размытие
изображений в формате RGBA8.

## Алгоритм

Для каждого пикселя вычисляется **взвешенное среднее** цветов всех соседей,
попадающих в круг заданного радиуса. Вес каждого соседа равен его евклидову
расстоянию до центрального пикселя (чем дальше сосед — тем сильнее влияет).
Центральный пиксель в усреднение не включается.

Алгоритм повторяется `iterations` раз, что позволяет получить более выраженный
эффект без увеличения радиуса.

## Параметры (`params.yaml`)

```yaml
radius: 5       # радиус круговой окрестности в пикселях (0 — без эффекта)
iterations: 1   # количество повторений алгоритма (0 — без эффекта)
```

Файл передаётся хосту через флаг `--params`. При некорректном YAML плагин
применяет значения `radius: 0, iterations: 0` (изображение не изменяется).

## Использование

```bash
# Сборка плагина
cargo build -p blur_plugin

# Применение через image_processor
cargo run -p image_processor -- \
  --input photo.png \
  --output blurred.png \
  --plugin blur_plugin \
  --params blur_plugin/params.yaml
```
